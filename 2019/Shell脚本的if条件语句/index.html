判断字符串是否为数字的思路。

  1. 使用sed加正则表达式

    判断思路：删除一个字符串中的所有数字，看字符串的长度是否为0，如果不为0，则说明不是整数。

  2. 变量的子串替换加正则表达式

    判断思路：如果num的长度不为0，并且把num中的非数字部分删除，然后再看结果是不是等于num本身，如果两者都成立，则num就是数字。
